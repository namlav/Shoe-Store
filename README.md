HOW TO USE
1. Clone repo về máy tính
2. Khởi chạy 'LTW.sql' trong SQL Server (lưu ý tên server là '.' hoặc người clone về tự đặt)
3. Khởi chạy 'WebGiayyy.sln' trong Visual Studio 2022
4. Build solution
5. Khởi chạy website (F5)
